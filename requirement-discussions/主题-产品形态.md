## 当前结论

### MVP 产品形态

- **形态**:Streamlit dashboard 4 模块(回放 / 实时教练 / AI 教练 / 设置)
- **非 HUD overlay** — 远期 #LR5 / #LR10 才上 PySide6 桌面 app
- **数据来源**:VPS PG(Tailscale 私有内网)
- **运行机器**:家里低配电脑 = dashboard only(不跑 pipeline);Win 桌面机 = pipeline only

### LLM 应用红线(收口)

- **唯一允许**:解释层(策略建议 / 牌局解释 / 自然语言推理)
- **绝对禁止**:运维 / 数据修复 / 规则化决策 / 自动写库
- 远期路线:#LR7-A 现成 Qwen2.5-7B-Instruct + #LR7-B LoRA 微调

### 打包路径

- **禁止 PyInstaller**(AV 误报严重)
- **采用 Inno Setup** + Python embeddable 包,小圈分发
- 仅自用 / 小圈,**不公开发布**(WePoker ToS 边界)

### 异步开发架构

- **Win 桌面机**:pipeline 跑 + 数据写 VPS PG
- **家里电脑**:dashboard 读 VPS PG + 前端开发
- **两端不冲突**:几乎不改同一文件
- **setup 文档**:`docs/setup_home_dashboard.md` 8 步 + 7 常见错误
