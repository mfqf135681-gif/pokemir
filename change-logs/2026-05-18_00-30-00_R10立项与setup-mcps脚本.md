## 7. 手动操作提醒

⚠️ **手动操作**：

1. **可选**：在 Linux dev / Windows 测试机配置 R-10 env vars（参考 `docs/dev-workflow.md §6.1`）。可放 `.envrc` 配合 `direnv`，或加到 venv activate 后的手动 export：
   ```bash
   export HF_HOME="$(pwd)/.cache/huggingface"
   export PIP_CACHE_DIR="$(pwd)/.cache/pip"
   export PYTHONPYCACHEPREFIX="$(pwd)/.cache/pyc"
   export NPM_CONFIG_CACHE="$(pwd)/.cache/npm"
   ```

2. **跑 setup-mcps.sh** 装 MCP（任选时机）：
   ```bash
   # Linux dev:
   bash tools/setup-mcps.sh   # postgres 步骤可直接 Enter 跳过（VPS 无 PG）

   # Windows 测试机:
   POKEMIR_FS_PATH=/c/pokemir-test bash tools/setup-mcps.sh
   ```
   装完后**重启 Claude Code session** 才能在我下一会话里看到 MCP 工具。

3. **若要装 semgrep MCP**：先在 venv 装 semgrep 本体（**禁用** pipx）：
   ```bash
   .venv/bin/pip install semgrep
   ```
   然后重跑 `bash tools/setup-mcps.sh`（脚本会检测到 semgrep 在 PATH 后装 MCP）。

4. **`~/.claude.json` 权限**：装完 postgres MCP（含 DSN）后核对：
   ```bash
   stat -c %a ~/.claude.json  # 应为 600
   ```
