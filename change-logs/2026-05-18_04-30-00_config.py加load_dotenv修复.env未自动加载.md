## 7. 手动操作提醒

⚠️ **手动操作**（R-8 间接影响的预防性提示）：

`load_dotenv()` 现在让 `.env` 中**所有** env var 真正生效。请核对 `.env` 中以下值是否符合预期：

```bash
grep -E "^POKEMIR_|^HF_" .env
```

特别注意：
- `POKEMIR_MODEL_DIR`（默认 `./models`，配合 R-10）
- `POKEMIR_EASYOCR_DIR`（默认 `${MODEL_DIR}/easyocr`，配合 R-10 治本）
- `HF_ENDPOINT`（默认 `https://hf-mirror.com`，国内镜像）
- `POKEMIR_API_HOST` / `POKEMIR_API_PORT`（默认 `127.0.0.1:8765`）
- `POKEMIR_LOG_LEVEL`（默认 `INFO`）

若 `.env` 中**没有**这些行，仍走 `config.py` default（与之前行为一致）。若**有但值错**，现在会立刻生效——必要时删除 `.env` 中错的行让 default 恢复。
