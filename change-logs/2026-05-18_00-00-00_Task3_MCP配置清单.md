⚠️ **手动操作**：

1. **Linux dev (本机) 安装 context7 + filesystem**（参考 `docs/mcp-setup.md §3.1 / §3.3`）：
   ```bash
   claude mcp add filesystem -s local -- \
     npx -y @modelcontextprotocol/server-filesystem /home/alxe/project/pokemir
   claude mcp add context7 -s user -- npx -y @upstash/context7-mcp
   claude mcp list  # 验证
   ```
   完整退出当前 Claude Code session 再启，下次 session 内会出现 `mcp__filesystem__*` / `mcp__context7__*` 工具
