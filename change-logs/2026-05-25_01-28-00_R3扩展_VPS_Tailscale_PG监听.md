## 7. 手动操作提醒

⚠️ **Win 用户接下来的操作**(由用户在 Win 上执行):

### A. 装 Tailscale Windows 客户端

1. 打开 PowerShell(普通用户即可,装机器时会弹 UAC):
   ```powershell
   # 方式 1: winget 一键
   winget install --id tailscale.tailscale -e

   # 方式 2: 官网下载 msi
   # https://pkgs.tailscale.com/stable/#windows  →  下载 tailscale-setup-x.x.x.msi 双击装
   ```

2. 装完任务栏右下角会出现 Tailscale 图标 — 右键 → **Log in**

3. 浏览器弹 OAuth 登录页 — **用 `mfqf135681@gmail.com`** 同账号登(必须与 VPS 同账号,否则不在同一 Tailnet)

4. 登录成功后,Tailscale 客户端弹"This device is now in your Tailnet" → 完成

### B. 验证联通

```powershell
# 看本机 Tailscale IP
tailscale ip -4
# 应输出形如 100.x.x.x

# ping VPS Tailscale IP
ping 100.101.105.46
# 应能通(0% loss)

# 测 PG 端口可达(若装了 psql)
psql -h 100.101.105.46 -p 5432 -U poker_user -d poker_assistant
# 提示输密码 → 输 POKEMIR_DB_PASSWORD → 应进入 psql shell
```
